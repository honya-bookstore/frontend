## Overview

[Honya Bookstore](https://github.com/honya-bookstore/frontend) is a specialized e-commerce platform designed to provide a focused and seamless shopping experience for book enthusiasts.

- **Focused Experience**: Unlike other generic marketplaces, Honya Bookstore is specialized towards latest books, categorization, and occasional promotions, giving our customers a favored shopping experience.

- **Modern Tech Stack**: Powered by **Next.js** for SSR and speed, together with **Tailwind** for a clean aesthetic User Interface.

## Features

Honya Bookstore comes along with a variety of features to make sure that each shopping experience is nothing but a journey through the world of books:

### **Discover All Books**: 
Browse an extensive catalog of the most recent and popular book titles.

![Book Catalog](https://i.ibb.co/WW5tb1sK/Screenshot-2025-12-22-141541.png)

### **Advanced Search & Filtering**
Find exactly what you are looking for.

- **Filter** by Genre, Author, Price Range, or Publication Date.
- **Sort** by Popularity, Newest Arrivals, or Price (Low/High).

![Search and Filter](https://i.ibb.co/hJNrwdQG/Screenshot-2025-12-22-140829.png)

### **Frequest Promotions**
Keep users engaged with special discounts and offers, you don't want to miss out on these deals!

![Promotions](https://i.ibb.co/BVYCtDcr/Screenshot-2025-12-22-140950.png)

### **Reliable Customer Service**
Integrate support features to assist users with issues or inquiries, keeping customer satisfaction is our job.

![Customer Service](https://i.ibb.co/N6p2DQ5S/Screenshot-2025-12-22-141122.png)

### **Keep Up With Latest Releases**
Stay updated with the newest book releases and upcoming titles in our blog page and articles. Perhaps you will find one or two that satisfy your reading taste?

![Blog Page](https://i.ibb.co/rKKdQYGn/Screenshot-2025-12-22-141322.png)

## Tech Stack

- **Next.js**: Powerful React framework with SSR.
- **Tailwind CSS**: CSS framework, UI have never been easier.
- **TypeScript**: Type-safe JavaScript for better code quality.
- **Vercel**: Free hosting and deployment platform.
- **Figma**: UI/UX design tool.
